'JS', 'UX', 'Design', 'SQL', 'Bootstrap', 'Jquery', 'SASS' O projeto "Movess" é uma landing page de uma loja fictícia de sofás luxuosos. Foi desenvolvido utilizando as linguagens de programação JavaScript, CSS e HTML, além de fazer uso do framework Bootstrap. A página tem como objetivo apresentar e promover os produtos da loja de forma atrativa e responsiva.
